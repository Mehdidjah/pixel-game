# Platform Adventure

A challenging 2D platformer game built with React, TypeScript, and Tailwind CSS.

## Features

- 7 progressively challenging levels
- Smooth player movement and physics
- Collectible coins
- Lava obstacles with different movement patterns
- Clean, modern UI

## How to Play

- Use **Arrow Keys** or **WASD** to move
- **Up Arrow** or **W** to jump
- Collect all coins to complete each level
- Avoid the lava!

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.

### Vercel Deployment

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy with default settings
